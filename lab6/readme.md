Efectul lui Y:
- cand Y este mic (de exemplu 0), inseamna ca numarul de clienti care au cumparat produsul este mic sau 0. Daca teta nu e la fel de mic, stiind ca Y=0, poate insemna ca n (numarul de clienti) este mic.
- cand Y este moderat (de exempu 5), inseamna ca numatul de clienti care au cumparat produsul este moderat. In functie de teta, daca teta este mare poate inseamna ca si n este moderat sau mai mare.
- daca Y este mare (de exemplu 10), inseamna ca numarul de clienti care au cumparat produsul este mare. Daca teta este mare, atunci n nu trebuie sa fie neaparat mai mare decat 10.

Efectul lui teta:
- cand teta e mic (de exemplu 0.2), inseamna ca avem o posibilitate mai mica de cumparare per client
- cand teta e mare (de exemplu 0.5), inseamna ca avem o posibilitate mai mare de cumparare per client.
